# Style Rules

## Visual Style

- Use blue-white academic style.
- Prefer white or light gray backgrounds.
- Use deep blue for page titles, title bars, and structural accents.
- Use red only for key conclusions or critical numbers.
- Keep the top title bar unified across content slides.
- Place the China University of Mining and Technology logo consistently in the upper right when the user provides or already has a logo in the deck.
- Keep margins and whitespace generous.

## Layout

- Each slide should communicate one main point.
- Use left-text/right-image, centered chart + lower conclusion, or two/three-column academic layouts.
- Use native editable diagrams when images are unclear.
- Use three-line tables for formal parameter/settings tables.
- Experiment result slides should use conclusion first + chart/table support.
- Conclusion slides should separate research conclusions from limitations and future work.

## Avoid

- Business report style and decorative card clutter.
- Internet UI dashboard style.
- Large gradients or decorative blobs.
- Meaningless English decoration words.
- Dense paragraphs.
- Unreadable screenshots.
- Topic-mismatched images.
- Copying reference-template content, images, data, or professional diagrams.
