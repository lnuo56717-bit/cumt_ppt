# Visual Check Prompt

Use this prompt for final PowerPoint visual QA.

## Required Steps

1. Open the PPT in PowerPoint using Computer Use when available.
2. Inspect each slide in edit or slideshow view.
3. Check:
   - page displays normally;
   - title location and font are consistent;
   - CUMT logo is clear, consistent, and does not cover title/page number;
   - body text is not dense;
   - Chinese body uses Songti, titles use Heiti, English/numbers use Times New Roman;
   - images are not distorted, blurry, too small, or unrelated;
   - tables are centered and readable;
   - experiment result numbers are visible;
   - no text overflow or element overlap;
   - no reference template placeholder or unrelated content remains.
4. Make only small visual adjustments.
5. Export every slide as PNG into a preview folder.
6. Write `visual_check_report.md`.
